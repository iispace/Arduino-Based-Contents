# PT100 with Arduino Uno + Liquid Crystal Display(LCD)
![image](https://user-images.githubusercontent.com/24539773/162856833-ae9f8c99-9017-44dc-bdee-9b0525193b2a.png)




Materials:
- PT100 (3 wires)
- MAX31865 Breakout board
- Arduino Uno
- 4P Terminal
