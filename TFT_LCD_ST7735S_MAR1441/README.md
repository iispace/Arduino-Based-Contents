# Wiring<br>
| Arduino UNO | TFT LCD(Software SPI) | TFT LCD(Hardware SPI) |
| :-: | :-: | :-: |
| GND | GND | GND
| 3.3V | VCC | VCC
|D13||CLK
|D11||SDI
|A5|CS|CS
|A4|RST|RST
|A3|RS|RS
|A2|SDI|
|A1|CLK|
|A0|LED|LED
